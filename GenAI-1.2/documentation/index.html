What's included

After downloading the files from Themeforest and unzip them to the local disk on your computer. Inside the extracted folder you will see three folders named Documentation, genai-build & genai-dev.

  • Documentation - the documentation for the GenAI template
  • genai-build - the compiled files that you can cusmomize with your content and uplaod to your server
  • genai-dev - all the source files of the template with compiled and production ready files for the developers and advanced users to customize even more.

General Customization

You can customize any .html files inside the genai-build directory to change your contents to make it on your own.


Change Placeholder Images

When you download the template files you will see we used placholder images instead of those originals we used in our preview link. To change with your desired images replace the placeholder images from assets/images directory.

Logo

We use Default logo as a image logo. you can replace the image with your own image. If you want to use text instead of logo, just remove the image and add your text inside .navbar-brand

CSS Files

All the css files are located in the assets/css folder of the template.
plugins.css - This file includes all 3rd party plugins used in the template.
style.css - Main stylesheet which includes all the styles of the template.

JavaScript Files

All vendor, 3rd party plugin and our custom JavaScript files are located in the assets/js folder.
plugins.js - This files includes all 3rd party plugins in the template.
main.js - Our custom JavaScript file. You can add your custom js codes there if you need to.

PHP Contact Form

Our contact form is totally functional, after submit form you can see form data on your E-mail. To get the emails to your email address, go to the assets/php and open the contact_email.php in a text editor. Then change the receipient email address to your email address. You can also change the email subject.

Developer Mode

For advanced users and developers, you can customize more the template by the following steps:

  • Make sure you have installed nodejs on your computer.
  • Install gulp-cli on your computer by run command npm install -g gulp-cli from your terminal (Mac/Linux) or Command prompt/Powershell (Windows).
  • Navigate through genai-dev folder & run command npm install for installing all of your package.
  • Run command npm run start to start this project on development server. -- this will build the template and store them to the build folder.
  • After completing development, run command npm run prod -- this will build the template and store them to the dist folder. This files are minified css and js, images are optimized and compact by removing unused css from the css files.
